
## Lelas anteckningar

Hela denna branch är egentligen bara ett test på om denna SPA skulle fungera som grund men den är baserad på node framework Express, denna är ganska basic (minimalistisk framework jämfört med React) och används främst för SPA och REST APIs. 

Jag har bakat in Ahmads Budget- och Dashboard.html i folder 'Views' som ska laddas fram när man kommer in på dom rätta "sidorna"
(Dashboard är egentligen "Home" page men bara för att testa har jag satt upp den som en separat flik)

Osäkert dock om vi kan/får köra en SPA med någon framework alls men fördelen med denna isåfall skulle vara möjligheten att kunna använda History Api för att kunna spara data mellan "sidorna"
